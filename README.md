# Sistema de Gestión de Libros Electrónicos

## Descripción
Sistema desarrollado en Python para administrar libros electrónicos mediante programación orientada a objetos.

## Funcionalidades
- Registro de libros
- Visualización de libros
- Gestión de usuarios
- Gestión de préstamos
- Manejo de errores

## Clases implementadas
- Libro
- Usuario
- Prestamo
